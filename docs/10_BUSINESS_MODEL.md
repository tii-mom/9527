# 10. 商业模式

## 核心商业闭环

```txt
广告主付费
  -> 平台展示 sponsor 广告
  -> 开发者主动查看
  -> 平台获得广告收入
  -> 一部分收入进入奖励池
  -> 用户获得积分
  -> 积分商城兑换权益
  -> 用户留存和传播
  -> 平台广告库存增加
```

## 收入来源

```txt
Sponsor slot
CPM 广告
CPC 广告
CPA 分成
积分商城差价
提现手续费
B2B reward API
第三方 CLI/IDE SDK 授权
```

## 单位经济模型

假设：

```txt
DAU = 10,000
人均每天广告机会 = 10
用户 opt-in 率 = 20%
有效完成率 = 70%
CPM = $20
```

计算：

```txt
月有效展示 = 10,000 × 10 × 20% × 70% × 30 = 420,000
月广告收入 = 420,000 / 1000 × $20 = $8,400
用户奖励池 40% = $3,360
平台毛利 = $5,040
```

## 奖励分配建议

```txt
用户奖励池：30%-50%
平台毛利：50%-70%
```

## 增长飞轮

```txt
更多开发者安装
  -> 更多等待时间广告库存
  -> 广告主愿意投放
  -> 奖励池变大
  -> 开发者更愿意使用
  -> 社区传播
  -> 更多开发者安装
```
